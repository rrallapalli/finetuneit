# Reproducibility note:
#   The two libraries that repeatedly broke on version drift are transformers
#   and trl. transformers is pinned to the version confirmed working on the
#   RunPod pod; trl is floored at the release where the current code's argument
#   names (processing_class, SFTConfig.max_length) are valid.
#
#   To FULLY lock this file, run on your known-good pod:
#       pip freeze | grep -Ei "transformers|^trl|peft|accelerate|bitsandbytes|datasets|unsloth" 
#   and replace the >= floors below with the exact == versions it prints.
#
#   torch is intentionally NOT listed: RunPod GPU templates ship a CUDA-matched
#   torch, and the setup script reuses it. Pinning torch here risks pulling a
#   wheel that mismatches the pod's CUDA driver.
#   unsloth is installed separately (setup script, --no-deps) to avoid it
#   changing torch; pin it there: pip install "unsloth==<your version>" --no-deps
# ---------------------------------------------------------------------------
